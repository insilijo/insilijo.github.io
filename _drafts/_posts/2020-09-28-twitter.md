A sample blog page that demonstrates the inclusion of Tweets/Timelines/etc.

# Tweet

An example of displaying a tweet:

<blockquote class="twitter-tweet" data-theme="light">
  <p lang="en" dir="ltr">Example tweet content would appear here. This is a placeholder for the Twitter embed.</p>
  &mdash; Example Account (@example) 
  <a href="https://twitter.com/rubygems/status/518821243320287232">September 28, 2020</a>
</blockquote>
<script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

# Timeline

An example of pulling from a timeline:

<div class="twitter-timeline-placeholder" style="max-width: 500px;">
  <p><strong>Twitter Timeline Example</strong></p>
  <ul>
    <li>Tweet 1: Example content from timeline</li>
    <li>Tweet 2: Another example tweet</li>
    <li>Tweet 3: Third example tweet</li>
  </ul>
  <p><em>Note: Twitter timeline embeds require the jekyll-twitter-plugin which is not available in GitHub Pages builds.</em></p>
</div>

# Additional Details

For more details on using the plugin visit: [jekyll-twitter-plugin](https://github.com/rob-murray/jekyll-twitter-plugin)
